Portfolio Grunblatt

Information : Ceci est un essai de webgl pour faire un petit portfolio avec une insertion de quelque projets.

Lancement :
Pour lancer le site vous devez: 
	Extraire le fichier 
	Télécharger node js avec https://nodejs.org/fr/ sauf si vous l'avez déjà
	Ouvrir shell 
	Rentrer la commande: Cd repertoire/portfolio_files/portfolio_cyp/
	Rentrer la commande: npm run dev
	Taper sur votre site web: localhost:3000
