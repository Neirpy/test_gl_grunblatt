Bienvenue sur mon portfolio.
Pour lancer celui-ci, il faut :
	Extraire le fichier dans un répertoire
	Ouvrir l'invite commande ou git bash ou shell
	Taper Cd le répertoire où se situe le fichier/portfolio
	Taper npm run dev
Il faut bien sûr avoir installer node.js au préalable, ici un lien pour l'installer :https://nodejs.org/en/
